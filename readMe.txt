Le Bot discord en Orienté objet se lance avec la ligne de commande "python main.py -c token", token est le json contenant l'api meteo et le token discord
Le bot contient les commande suivante :

-!Hello : qui renvoi salut
-!Image : qui renvoi une image au hasard
-!Meteo + [ville] : qui renvoi la meteo et l'icone correspondant de la ville en question
-!Nombre: qui renvoi un nombre aleatoire compris entre 0 et 100
-!Telegram : cree une discussion privée en envoi le lien d'une channel Telegram
-!Help : envoi un resumé des commande
